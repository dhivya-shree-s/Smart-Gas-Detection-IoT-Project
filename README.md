This project, selected for the **Naan Mudhalvan scheme**, is an **IoT-based solution** designed to enhance household safety by automating gas leakage detection and simplifying the LPG cylinder refill booking process.

* **IoT Integration:** Utilizes the **ESP8266** microcontroller for Wi-Fi connectivity.
* **Smart Gas Booking:** Integrates a load cell/weight sensor to determine gas level and automatically or manually initiate a refill request via a **Python backend**.
